# Time Series Lab

Analyse Time Series data in your browser. 


# Working with the project

Install Rust and WASM tools:

https://rustwasm.github.io/docs/book/introduction.html
 
Build Rust backend: 
```sh
$ wasm-pack build
```

Init npm package
```sh
$ cd www
(www)$ npm install
```

Run local server
```sh
(www)$ npm run server
```

Deploy application for histing on Github
```sh
(www)$ npm run deploy
```
